# Le Chef — команда агентов

Шесть субагентов, каждый отвечает за своё направление. Запускаются из Claude Code
командой /agents или автоматически, когда задача подходит под описание агента.

| Агент | Направление |
|---|---|
| app-architect | Архитектура, навигация, каркас |
| backend-dev | API, БД, распознавание фото/голоса |
| frontend-dev | Экраны, компоненты, состояние |
| recipe-curator | Сбор и структурирование рецептов |
| competitor-scout | Анализ похожих приложений |

Рекомендованный стек: React Native + Expo (фронт), FastAPI + PostgreSQL (бэк).
Контракт между бэком и фронтом — docs/api-contract.md. Навигация — docs/navigation.md.
